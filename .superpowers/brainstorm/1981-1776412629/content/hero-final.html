Design notes

Active session redirect
If a session with status: "active" exists, this page is never shown — router redirects immediately to /log/:sessionId.
No sidebar on this route
The intro page uses its own full-screen layout. The collapsible sidebar appears on all other routes (/log, /exercises, etc.).
Green glow — CSS only
Two radial-gradient blobs: top-left (stronger, 12% opacity) and bottom-right (softer, 6%). No images needed.
Design token source
Bootstrap with npx getdesign@latest add spotify. Tokens replace the current TanStack starter CSS variables.
